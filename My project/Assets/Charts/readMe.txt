If you want make rhythm game's charts

If you want to repeat this bar, write '@' and number of repetition(can skip). ex) @1 == @

You should write BPM and number of repetitions on first line. ex) 300 5
And you write number of repetitions lines.
This line is real chart. 
You write chart by following this rules
	1. emtpy line is '.'. ex) ....
	2. single note is 's'. ex) ...s
	3. long note is 'l'. ex) l...

And you can control this line's rhythm 
	1. mismatched is to add line's end '*'. ex) ...s *
	2. rest beat is to add line's end '#' and number of rest beat(can skip). ex) ...s #1 == ...s #
